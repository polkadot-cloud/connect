# Connect Vault

Polkadot Vault (QR-based) wallet adaptor for @polkadot-cloud/connect

## Installation

```bash
npm install @polkadot-cloud/connect-vault
```

or

```bash
yarn add @polkadot-cloud/connect-vault
```

or

```bash
pnpm add @polkadot-cloud/connect-vault
```

## Usage

```typescript
import { /* your imports */ } from '@polkadot-cloud/connect-vault'
```
