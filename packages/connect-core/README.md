# Observables Connect

Observables for extensions and accounts discovery

## Installation

```bash
npm install @polkadot-cloud/connect-core
```

or

```bash
yarn add @polkadot-cloud/connect-core
```

or

```bash
pnpm add @polkadot-cloud/connect-core
```

## Usage

```typescript
import { /* your imports */ } from '@polkadot-cloud/connect-core'
```

### Observables

This package provides RxJS observables for reactive programming:

```typescript
import { /* specific observable */ } from '@polkadot-cloud/connect-core'

// Subscribe to observables
/* your observable */.subscribe(value => {
  console.log(value)
})
```
